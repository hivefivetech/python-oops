## 📚 About python-oops

python-oops is a simple and clear collection of examples for learning Python's Object-Oriented Programming (OOP) concepts. It covers key ideas like encapsulation, inheritance, polymorphism, abstraction, and object creation patterns. This project organizes these topics in an easy-to-understand way. You can explore how these principles work in Python using code examples.

If you want to understand how Python handles objects and classes, this project can help. It is designed for beginners who want to learn step-by-step, without prior programming knowledge.

## 🔍 What You Will Learn

python-oops demonstrates these core Python OOP ideas:

- **Encapsulation:** How to keep data safe inside classes.
- **Inheritance:** How classes can reuse code from other classes.
- **Polymorphism:** How objects can behave differently using the same interface.
- **Abstraction:** Using simple interfaces to hide complex logic.
- **Object Creation Patterns:** How to create objects using design patterns like the Factory or Singleton patterns.

Each concept comes with clear code examples. This helps you see how OOP works and why it matters for organizing software.
